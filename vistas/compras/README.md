# Módulo de Compras

## Propósito
Directorio destinado a contener las vistas y funcionalidades del área de Compras. Actualmente se encuentra en una etapa inicial de desarrollo o como un contenedor en estructuración.

## Archivos
- `vista_compras.php`: Archivo principal o esqueleto de la vista de compras.

## Tablas Principales Usadas (Proyectadas)
- `scmp`: Maestro/cabecera de compras.
- `sprv`: Maestro de proveedores.
