# Módulo de Inventario de Hardware

## Propósito
Este directorio contiene el Dashboard de Inventario de Hardware del departamento de TI/Sistemas. Su función es registrar, listar y auditar los equipos tecnológicos de la empresa, su asignación y estado operativo.

## Archivos
- `vista_inventario_hardware.php`: Interfaz principal para visualizar y gestionar la tabla de hardware.

## Tablas Principales Usadas (Base de Datos)
- `invsis`: Tabla principal de registro del inventario de sistemas y hardware.
